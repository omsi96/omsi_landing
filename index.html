iOS Development Course

By Omar Alibrahim

Join us if you would like to have HANDS-ON workshops to learn iOS app development. We offer two types of courses which targets beginners and intermidiate level iOS developers. First course is 5-days course (Starts on Jan. 19th) that does not need any prerequisites which going to teach you how to develop iOS apps from scratch. The second course is 5-days course (Starts on Jan. 29th) that requires first course which going to teach you how to connect your phone to a back-end server, and advanced topic in iOS architecture.

What you
Will Learn

After the 2 courses, you will be able to create and design your own iOS apps by yourself. The following are the main points of what you will learn in both courses.

Swift Langauge

Swift is a new powerful langauge introduced to make powerful apps on iOS. In the first course, we will learn how to program with Swift from scratch. No previous knowledge is required. In the second course, we will go even further with Swift, and will dig deeper with advanced Swift featured, such as Generics, Protocols, Enums, Closuers and others.

iOS app development

In the first course, we will learn how to make iOS apps along with learning Swift using a UIKit library. You will be able to create amazing apps like X-O game, Athkar app, Bmi Calculator and much much more. See examples section to see previous students applications.

Connecting apps to a server

In the second course, we will learn how to connect our apps to a server and a database to create a social networking app with Firebase.

Requirements

To attend this workshop, you need to have any Apple Mac machines (MacBook, MacBook Pro, or MacBook Air) and you need to install the latest version of Xcode on your machine before attending the workshop.
