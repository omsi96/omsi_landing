What you
Will Learn

You will be able to create and design your amazing iOS apps by yourself. The following are the main points of what you will learn in both courses.

Swift Langauge

Swift is a new powerful langauge introduced to make powerful iOS apps. In the first course, we will learn how to program using Swift language from scratch. No previous knowledge is required. In the second course, we will go even further with Swift, and will dig deeper into advanced Swift features such as Generics, Protocols, Enums, Closuers and others.

iOS App Development using SwiftUI

You will learn the fundementals of the new released framework, SwiftUI. SwiftUI is a new way to create amazing iOS apps with simple lines of codes. SwiftUI depends heavily on advanced Swift features. In this course, we're going to cover most of SwiftUI features that enables you to build awesome apps.

Connecting Apps to a Server

You will learn how to make your app interactive by sending and recieving data from a database on the cloud using firebase.
